# FabricStore — Brand Identity Guidelines

**Version:** 1.0  
**Date:** 2026-07-10  
**Scope:** Customer-facing store + Internal dashboard  

---

## 1. Brand Foundation

### 1.1 Brand Essence

FabricStore exists at the intersection of **craftsmanship** and **commerce**. The brand is warm, tactile, and trustworthy — reflecting the rich textile traditions of Nigeria while delivering a modern, reliable e-commerce experience. We make buying fabric as effortless as the fabric itself is beautiful.

### 1.2 Mission Statement

To connect fabric lovers — locally and globally — with the perfect fabric for every occasion, through a seamless platform that respects the craft, honours the customer, and empowers the business.

### 1.3 Vision Statement

To be the most trusted destination for African fabric commerce, where every thread tells a story and every order strengthens a community of makers, buyers, and creators.

### 1.4 Core Values

| Value | Description |
|---|---|
| **Craftsmanship** | We honour the art of fabric-making. Every product detail, colour swatch, and description reflects the quality of the material. |
| **Trust** | Customers trust us with their orders and payments. Staff trust the system to be fair and transparent. The Owner trusts the data. |
| **Clarity** | No hidden fees, no confusing stock, no guesswork. What you see is what you get, down to the exact colour and metre. |
| **Connection** | Bridging Nigerian textile artistry with the world, and connecting the team to a single source of truth. |
| **Growth** | Built to scale — more fabrics, more customers, more insights, more countries. |

### 1.5 Brand Personality

| Trait | Expression |
|---|---|
| **Warm** | Welcoming tone, rich colours, human-centred design |
| **Premium** | Polished UI, high-quality imagery, refined typography |
| **Reliable** | Clear status, predictable behaviour, transparent pricing |
| **Modern** | Clean layouts, minimal clutter, fast interactions |
| **Rooted** | Celebrates African textiles and Nigerian business culture |

## 2. Brand Voice & Tone

### 2.1 Voice Principles

| Principle | Explanation |
|---|---|
| **Approachable, not casual** | Friendly but professional. We say "you" and "we", not "one". |
| **Descriptive, not flowery** | Fabrics sell themselves. Describe honestly. Let the images speak. |
| **Clear, not cold** | Use plain language. Avoid jargon. Especially in checkout and order tracking. |
| **Helpful, not pushy** | Guide the customer. Never upsell in a way that feels manipulative. |
| **Confident, not arrogant** | We know our fabrics. We know our process. We state facts. |

### 2.2 Voice Guidelines by Context

| Context | Tone | Example |
|---|---|---|
| **Landing Page** | Inspiring, warm | "Find the fabric that tells your story." |
| **Product Descriptions** | Descriptive, precise | "Premium French Lace — delicate floral pattern on soft mesh. 5 yards per piece." |
| **Checkout** | Clear, reassuring | "Your order is protected. Pay securely with Paystack." |
| **Order Confirmation** | Celebratory, warm | "Your fabric is on its way! Here's your order summary." |
| **Order Tracking** | Informative, calm | "Status: Packing. Your order is being carefully prepared." |
| **Dashboard (internal)** | Neutral, functional | "3 orders need attention. 2 colour variants are low in stock." |
| **Error Messages** | Helpful, not blaming | "We couldn't process this payment. Please try a different method or contact support." |

### 2.3 Words to Use & Avoid

| Use | Avoid |
|---|---|
| Fabric, textile | Cloth (too generic) |
| Colour variant, shade | Colour option, colour choice |
| Per yard, per metre | Per piece (ambiguous) |
| Order ID | Reference number, ticket |
| Pay on Delivery | Cash on delivery (POD is clearer) |
| One-time code | OTP, verification code |
| Fabric type | Category, group |

## 3. Logo & Wordmark

### 3.1 Logo Concept

The FabricStore logo combines a stylised **spool of thread** (forming an abstract 'F') with the wordmark **FabricStore** in a refined sans-serif typeface. The thread element subtly resembles the warp/weave of fabric when inspected closely.

**Symbol:** A minimal, geometric icon — two interlocking arcs suggesting both a spool and the 'F' letterform. The arcs represent connection (local to global, buyer to maker).

### 3.2 Logo Variations

| Variation | Use Case |
|---|---|
| **Full Logo** (symbol + wordmark) | Landing page header, website header, printed materials |
| **Symbol Only** (icon alone) | Favicon, app icon, social media avatar, mobile header (space-constrained) |
| **Wordmark Only** (text alone) | Footer, secondary pages, email signatures |

### 3.3 Logo Clear Space

Minimum clear space around the logo equals the height of the 'F' in FabricStore. No other elements should intrude within this boundary.

### 3.4 Logo Minimum Sizes

- **Full logo:** 160px wide (digital), 1.5 inches (print)
- **Symbol only:** 32px (favicon), 24px (inline)
- **Wordmark only:** 120px wide

### 3.5 Incorrect Logo Usage

- Do not stretch or distort proportions
- Do not change colours from the brand palette
- Do not apply drop shadows or effects
- Do not place on low-contrast backgrounds
- Do not rotate or angle the logo
- Do not rearrange symbol and wordmark positions

## 5. Typography

### 5.1 Primary Typeface (Customer Store)

**Playfair Display** (Headings) — serif, elegant, reflects textile craftsmanship  
**Inter** (Body) — sans-serif, highly readable, modern

### 5.2 Primary Typeface (Dashboard)

**Inter** (All text) — functional, compact, information-dense

### 5.3 Font Stack

```css
/* Customer Store */
--font-heading: 'Playfair Display', Georgia, 'Times New Roman', serif;
--font-body: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;

/* Dashboard */
--font-dashboard: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;
```

### 5.4 Type Scale

| Level | Size | Weight | Line Height | Usage |
|---|---|---|---|---|
| **Display XL** | 48px / 3rem | 700 (Bold) | 1.1 | Hero headings, landing page |

| **Heading 1** | 32px / 2rem | 700 | 1.25 | Section headings |
| **Heading 2** | 24px / 1.5rem | 600 (Semi-Bold) | 1.3 | Subsection headings |

| **Body** | 16px / 1rem | 400 | 1.6 | Default body text |
| **Body Small** | 14px / 0.875rem | 400 | 1.5 | Captions, metadata |

| **Label** | 14px / 0.875rem | 500 (Medium) | 1.4 | Form labels, button text |

### 5.5 Dashboard Type Scale (Compact)

| Level | Size | Weight | Usage |
|---|---|---|---|
| Dashboard H1 | 24px / 1.5rem | 600 | Page title |
| Dashboard H2 | 18px / 1.125rem | 600 | Section title |
| Dashboard Body | 14px / 0.875rem | 400 | Table content, forms |
| Dashboard Body Small | 12px / 0.75rem | 400 | Metadata, timestamps |
| Dashboard Label | 12px / 0.75rem | 500 | Table headers, form labels |

### 5.6 Typography Rules

- Maximum line length for body text: 75 characters (store), unlimited (dashboard)
- Heading letter-spacing: -0.01em (tighten for large sizes)
- Body text letter-spacing: normal
- Never use all-caps for body text — reserved for badges, labels, and buttons
- Dashboard text should be dense but not cramped — aim for 1.4 line-height minimum

## 6. Imagery & Photography

### 6.1 Product Photography Style

| Attribute | Standard |
|---|---|
| **Lighting** | Natural, diffused — evenly lit fabric showing true colour |
| **Background** | Clean white or light grey (consistency across catalogue) |
| **Composition** | Flat lay (fabric spread flat) and draped (showing fall/texture) |
| **Resolution** | Minimum 1200px on longest side; zoom-capable (2000px+ recommended) |
| **Format** | JPEG (product), WebP (web delivery with fallback) |
| **Colour Accuracy** | Colour-calibrated — the hex value in the database should match the image closely |
| **Consistency** | Same lighting, angle, and styling across all products |

### 6.2 Image Types Needed Per Product

| Image Type | Purpose | Count |
|---|---|---|
| **Primary** | Main catalogue thumbnail and detail hero | 1 |
| **Swatch Close-up** | Texture/detail view | 1-2 |
| **Colour Variant** | Each colour variant shown on the fabric | 1 per variant |
| **Lifestyle** | Fabric styled (dress, decor, etc.) — optional | 1-2 |

### 6.3 Brand Photography (Non-Product)

- **Hero / Landing:** Warm, lifestyle imagery — fabrics in use, hands working with fabric, finished garments
- **About:** Behind-the-scenes — sourcing, workspace, team
- **Testimonials:** Customer photos (optional, with permission)
- **Tone:** Authentic, not overly produced — real people, real fabric

### 6.4 Illustration Style

- **Style:** Minimal line art with warm undertones, using Deep Indigo and Warm Gold
- **Usage:** Empty states (cart, order history), illustrations for About/Hero, decorative elements
- **Tone:** Elegant but approachable — avoid overly abstract or corporate styles

## 7. Iconography

### 7.1 Icon Style

- **Set:** Phosphor Icons or Lucide Icons (consistent outline style)
- **Weight:** Regular (1.5px stroke) for store; Bold for dashboard emphasis
- **Size:** 24px default (store), 20px default (dashboard inline), 16px for inline with text
- **Colour:** Inherits text colour or Deep Indigo for standalone icons
- **Custom Icons (brand-specific):**
  - Fabric type icons (Lace, Satin, Ankara, etc.) — small illustrations
  - Colour swatch selector — circular filled with hex colour, white border
  - Payment method icons (bank, card, POD)

### 7.2 Icon Usage Rules

- Always pair with text labels for navigation (never icons alone for critical navigation)
- Loading states: use the Phosphor Spinner icon, animated
- Social media icons: full-colour brand versions for footer

## 8. Brand Applications

### 8.1 Customer Store Visual Style

- **Layout:** Clean, generous whitespace, card-based product display
- **Cards:** White background, subtle border, rounded corners (8px), gentle shadow on hover
- **Navigation:** Fixed top bar, Deep Indigo background, white text, Warm Gold active indicator
- **Footer:** Deep Indigo background, white text, three-column layout (links, contact, social)
- **Checkout:** Minimal, distraction-free — remove nav, focus on completion
- **Mobile:** Bottom navigation bar, full-width cards, hamburger menu for filters

### 8.2 Dashboard Visual Style

- **Layout:** Sidebar navigation (collapsible, 240px expanded), main content area, top bar with search + user menu
- **Sidebar:** Charcoal background, white text, active item highlighted with Warm Gold left border
- **Data Tables:** Striped rows (white/Off White), sticky headers, sortable columns with visible indicator
- **Cards:** White, no border (flat), subtle shadow, 8px radius
- **Buttons:** Compact (32px height for default), label in Sentence Case
- **Modals:** Centered, overlay backdrop at 50% opacity Charcoal, close on Escape and backdrop click

### 8.3 Email Communications

- **Layout:** Single column, max 600px width, centred
- **Header:** FabricStore logo + Deep Indigo divider
- **Body:** White background, Charcoal text, Inter font
- **CTAs:** Warm Gold button, white text
- **Footer:** Small text, contact info, unsubscribe link (where applicable)
- **Transactional emails:** Order confirmation, payment receipt, shipping update, welcome

### 8.4 Social Media Profile

- **Avatar:** FabricStore symbol (icon only) on Warm Gold circle background
- **Cover Photo:** Lifestyle fabric imagery with brand gradient overlay (Deep Indigo to Burgundy)
- **Bio Tone:** Friendly, informative — "Premium fabrics for every occasion. Shop online — Nigeria & International."

## 11. Brand Guidelines Compliance

### 11.1 Do's

- Use Deep Indigo as the dominant colour (60% of colour surface)
- Use Warm Gold sparingly for emphasis (10% of colour surface)
- Maintain generous whitespace around the logo
- Use high-quality, consistent product photography
- Apply the voice principles to all customer-facing copy

### 11.2 Don'ts

- Never use bright neon or pastel colours outside the approved palette
- Never pair Deep Indigo with a similarly dark colour (loss of contrast)
- Never stretch, pixelate, or alter the logo
- Never use Comic Sans, Papyrus, or decorative display fonts
- Never place text directly over busy fabric imagery without a scrim overlay
- Never use stock photography that looks generic or inauthentic

## 12. Dashboard Branding (Internal)

### 12.1 Brand Presence

- Dashboard uses the FabricStore logo (wordmark) in the sidebar header
- Colour palette is a subdued version of the brand: Charcoal sidebar, Deep Indigo accents, Warm Gold sparing for alerts
- No Playfair Display — Inter only (functional density)
- No lifestyle imagery — data focus
- Success/Warning/Error semantic colours match the brand palette

### 12.2 Dashboard Tone

- **Neutral, functional, professional**
- No marketing language — use clear, concise labels
- Staff names displayed with first name only in activity logs (UX comfort)
- Status badges use brand semantic colours: Green (confirmed, delivered), Amber (pending, low stock), Red (cancelled, out of stock), Blue (in progress)
